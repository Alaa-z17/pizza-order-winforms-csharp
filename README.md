# Pizza Order System – WinForms + JSON

[Video Walkthrough](https://youtu.be/...)

## Features
- Order pizza with toppings, size, crust, quantity
- Multiple orders (MDI)
- Order countdown timer
- JSON persistence

## OOP Concepts Demonstrated
- Classes, Encapsulation, Inheritance, Static Members, Enum

## WinForms Concepts Demonstrated
- CheckBox, RadioButton, ComboBox, GroupBox, MessageBox, PictureBox, ErrorProvider, NumericUpDown, Timer, MDI

## Build & Run
- .NET 10 required
- Open `PizzaOrderWinForms.sln` in Visual Studio → Build → Run
